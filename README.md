# DogApp

Dog search by breed name. Uses [Dog API](https://dog.ceo/).

## [Live Demo](http://test13378.futurehost.pl/dogapp/)

## Install dependencies

```
yarn install
```

## Develop

```
yarn start
```

## Test
```
yarn test
```

## Build

```
yarn build
```
